# E-Commerce Executive Analytics Platform

## Project Overview

This project analyzes sales performance, customer behavior, delivery operations, seller performance, and payment trends using the Olist Brazilian E-Commerce dataset.

The objective is to help business stakeholders monitor key performance indicators and make data-driven decisions.

## Business Problem

E-commerce companies need to understand:

* Revenue performance
* Customer purchasing behavior
* Product performance
* Delivery efficiency
* Seller contribution
* Customer satisfaction

This dashboard provides actionable insights through interactive visualizations and KPIs.

## Tools Used

* Power BI

## Key Insights

* São Paulo generated the highest revenue among all states.
* A small number of product categories contributed the majority of sales.
* Credit Card was the most preferred payment method.
* Most customers provided positive ratings.
* Delivery performance varied across states.
* A few sellers generated a significant portion of total revenue.

## Recommendations

* Focus marketing efforts on top-performing product categories.
* Improve logistics performance in states with higher delivery times.
* Strengthen customer retention strategies in key revenue-generating regions.
* Build partnerships with top-performing sellers.
* Improve customer experience to maintain high review ratings.
